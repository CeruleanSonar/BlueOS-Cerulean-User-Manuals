Additional Connection Possibilities

There are many possibilities for interconnecting the DVL and using it to control the ROV and/or monitor the DVL passively without interfering with ROV control. Three are shown below.
Using Both Interfaces. Serial and Ethernet interfaces can be used simultaneously for listening. We do not recommend sending commands on both interfaces. Pick one or the other.
Using a Port Replicator to monitor the DVL while also using it in ROV control.
In Body Frame mode, connections are simple and an optional on-ROV GPS would provide no benefit.
