Resetting the Communications Parameters to Factory Default

If you have set the serial port baud rate to a value your host cannot use, or if you aren’t sure what you’ve done to the Ethernet addressing, you can set the communications parameters back to factory default. This is done by holding down the Boot button on the electronics stack for at least two seconds (this is done while the system is powered up).
The Master CPU Awake red LED will turn off when you first press the button. If you release the button before two seconds, the LED will turn back on and normal processing will resume. If you continue to hold the button, the LED will begin flashing rapidly. At this point, you can release the button and the system will reboot with the reset parameters.
