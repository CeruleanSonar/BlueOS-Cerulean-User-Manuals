SET-SPEED-OF-SOUND

The SET-SPEED-OF-SOUND command sets the speed of sound in the body of water where the DVL is operating. The DVL accepts speeds between 1,100 and 1,950 m/S, and the value is stored in flash memory until a new value is received.
Example:
SET-SPEED-OF-SOUND 1500
