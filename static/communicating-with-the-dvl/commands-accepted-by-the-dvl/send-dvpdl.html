SEND-DVPDL

The SEND-DVPDL command turns the $DVPDL output message sending process on and off. The effect persists across power cycles. The parser is more lenient with the “on” message than the “off” message.
Examples:
SEND-DVPDL ON
SEND-DVPDL OFF
