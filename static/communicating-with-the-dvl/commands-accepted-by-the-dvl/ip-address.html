IP-ADDRESS

The IP-ADDRESS command stores or erases a static IP address for the DVL for use during the next boot-up. Setting an address of 0.0.0.0:0 erases any current static IP address, and the DVL will attempt to use DHCP to get an IP address next boot-up, resorting to the fallback address if DHCP fails. Note, the IP broadcast address 255.255.255.255 cannot be used as a static IP address for the DVL.
Examples:
IP-ADDRESS 192.168.2.3
IP-ADDRESS 0.0.0.0
