FALLBACK-ADDRESS

The FALLBACK-ADDRESS command stores a static IP address for the DVL for use during the next boot-up and is only used if there is no static IP address set and DHCP fails. Setting an address of 0.0.0.0 or 255.255.255.255 as a fallback address is not allowed. The DVL ships from the factory with the fallback address set to 192.168.2.2.
Example:
FALLBACK-ADDRESS 192.168.2.3
