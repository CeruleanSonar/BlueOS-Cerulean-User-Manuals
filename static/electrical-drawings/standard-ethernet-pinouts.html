Standard Ethernet Pinouts

Ethernet Pinouts for JST-GH connects which are found on the DVL Electronics Stack and some Ethernet switches designed for ROVs
RJ-45 (regular connector for CAT5/CAT6 Ethernet) for switches and hosts that DO support 24V POE (remember, 24V POE is required, not 48V). You can use either Type A or Type B pinouts.
RJ-45 (regular connector for CAT5/CAT6 Ethernet) for switches and hosts that DO NOT support 24V POE. You can use either Type A or Type B pinouts.
