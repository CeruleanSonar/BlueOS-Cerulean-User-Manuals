DVL-75 User Manual Overview

This system is designed for low-cost, medium accuracy measurement of the relative velocity between the sensor head and other objects. The system provides velocity outputs relative to the World Frame of reference and can estimate positions using dead reckoning.
This system is designed and priced for use by people who need to know generally where their ROV is located and/or want to hold a position in the face of currents or tether pull. It is not a survey-grade instrument.
